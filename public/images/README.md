# Project Images

Place your project images in this folder and reference them in `public/data/projects.json` as:

```json
"image": "/images/your-image-name.jpg"
```

## Recommended Specifications

- Format: JPG or PNG
- Dimensions: 800x500 pixels (or similar 16:10 ratio)
- File size: Under 500KB for fast loading
- Quality: 80-90% compression
